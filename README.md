# Prompt Forge

Prompt Forge is a minimal builder workspace for shaping prompts from a starting context into a clear target outcome.

The first screen is intentionally quiet: a black canvas, a centered empty state, and a floating composer for the first instruction.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
npm run lint
```

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- lucide-react
