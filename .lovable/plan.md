# افزودن سبک‌های ویدئوی توضیح‌دهنده (Explainer Styles)

## هدف
افزودن مجموعه سبک‌های ویدئوی توضیح‌دهنده‌ای که کاربر فهرست کرده، به بخش «تمپلیت‌های ویدئویی» در دیالوگ Product Ad Scenario، تا هنگام ساخت سناریوی تبلیغاتی قابل انتخاب باشند.

## محل تغییر
فقط فایل `src/modules/generator-ui/components/ProductAdDialog.tsx` (آرایه `VIDEO_TEMPLATES`). این موارد به‌صورت خودکار در همان UI رندر می‌شوند (گروه‌بندی، انتخاب و تزریق prompt به سناریو از قبل وجود دارد) — نیازی به تغییر منطق نیست.

## یک گروه جدید
افزودن یک گروه تازه با عنوان «ویدئوی توضیح‌دهنده / آموزشی» (Explainer & Educational) در کنار گروه‌های موجود، تا این سبک‌ها از سبک‌های فعلی «انیمیشن و موشن گرافیک» جدا و مرتب باشند.

عنوان گروه به ۶ زبان موجود (en, fa, ar, tr, es, fr) لوکالایز می‌شود — مطابق الگوی ثابت `Loc`.

## سبک‌هایی که اضافه می‌شوند
هر کدام با لیبل ۶‌زبانه، آیکون ایموجی و یک prompt انگلیسی توصیفی (هم‌سبک ورودی‌های فعلی):

| سبک | آیکون |
|---|---|
| Whiteboard Animation (وایت‌برد) | ✍️ |
| Blackboard Animation (بلک‌برد) | 🟢 |
| Glassboard Animation (گلس‌برد نئونی) | 🟩 |
| Line Art Animation (انیمیشن خطی) | 〰️ |
| Infographic Animation (اینفوگرافیک متحرک) | 📊 |
| 2D Flat Animation (طراحی تخت) | 🟦 |
| Isometric Animation (ایزومتریک) | 📐 |
| 2D Character Animation (کاراکترمحور) | 🧍 |
| Cut-out Animation (کات‌اوت کاغذی) | ✂️ |
| Stop Motion (استاپ موشن) | 🧩 |
| Screencast / UI Animation (اسکرین‌کست/رابط کاربری) | 🖥️ |
| Live-Action + Tracked Graphics (لایو اکشن با المان شناور) | 🎞️ |

نکته: سبک‌های «تایپوگرافی متحرک»، «موشن کمیک» و «ویدئوی توضیحی (Explainer)» از قبل در گروه انیمیشن موجودند؛ برای جلوگیری از تکرار، تکراری اضافه نمی‌شوند و فقط موارد جدید در گروه تازه قرار می‌گیرند.

## جزئیات فنی
- هر آیتم با ساختار `VideoTemplate` ({ id, label: Loc, icon, group, prompt }) افزوده می‌شود.
- prompt هر سبک، توصیف کارگردانی/ویژوال آن سبک به انگلیسی است تا به موتور تولید سناریو پاس داده شود.
- چون `SCENE_GROUPS`/گروه‌بندی به‌صورت پویا از داده‌ها ساخته می‌شود، گروه جدید خودکار در UI ظاهر می‌شود.

## تأیید
پس از اعمال، در دیالوگ Product Ad، در بخش «تمپلیت‌های ویدئویی» یک دسته جدید با ۱۲ سبک توضیح‌دهنده دیده و قابل انتخاب خواهد بود.
